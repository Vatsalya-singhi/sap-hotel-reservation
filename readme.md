# SAP- Hotel Reservation

Hotel Reservation is a typescript application with suitable test cases.

## Installation

Use the package manager [node](https://nodejs.org/en/) to install dependencies.

```bash
npm install
```

## Usage

```python
npm run start
```
```python
npm run test
```
